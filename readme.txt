== Description ==
<p>A powerful Facebook and Twitter widgets integration that allows you to display facebook (timeline, events and messages), twitter follow button and twitter timeline for your wordpress website.</p>
<p>This plugin is very useful for displaying facebook page like button, twitter follow button and feeds. It will improve page likes and followers.</p>
<p>Just activate and setup your facebook and twitter widgets from Appearance >> Widgets section.</p>

<h4>Features</h4>
<pre><code>
* Increase your facebook page likes and twitter followers
* Display facebook timeline/events/messages
* Twitter feeds integration
* Add/Remove Twitter follow button with its customizations
* Easy to setup and maintain
* Clean & simple widgets options
</code></pre>

== Installation ==
1. Put the folder named "advanced-facebook-twitter-widget" under '/wp-content/plugins/' directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress. For more details, http://codex.wordpress.org/Managing_Plugins
3. Goto Appearance -> Widgets.
4. For facebook timeline use "Facebook Page Like and Feeds" and for twitter follow button and timeline use "Twitter Tweets With Follow Button" widgets.
5. Add widgets into the sidebars or any customized section in Widgets area and it will be working fine.
